# frontend (React + TypeScript)

This folder will contain the React + TypeScript dashboard.

Responsibilities:
- Upload source and target JSON
- Show AI mapping suggestions
- Allow manual overrides
- Preview transformation result

Next steps:
1. Run `npx create-react-app frontend --template typescript` (or Vite) inside this folder.
2. Create UI components for uploader, mapping editor, preview.
3. Connect to mapping-service and validation-service endpoints.
